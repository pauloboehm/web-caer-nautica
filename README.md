# web-caer-nautica

## Serviços e eventos

Site da seção náutica do Clube da Aeronáutica de Brasília

## Rastreador

Gera um arquivo GPX com pontos a cada 1 segundo para uso nas competições de rally.
_rastreador.html_ usa navigator.watchposition para acompanhar o percurso.
_rastreadorI.html_ usa navigator.getposition com setinterval para buscar a posição a cada segundo.
_rotacaer.html_ desenha um caminho para ser seguido, ao mesmo tempo que grava o percurso.
